# Phase 3 — Module Français
**Studio Étoile | Phase 3**
**Niveau : 5e–6e année, Québec (PFEQ)**
**Durée estimée : 3 semaines**

---

## Objectif de la Phase

Développer les compétences en langue française à travers :
- Lecture compréhensive engageante (textes à thème beauté/mode)
- Rédaction créative via le **Journal de Beauté**
- Grammaire et orthographe intégrées au contexte
- Vocabulaire enrichi naturellement

---

## Compétences Ciblées (Programme QC 5e–6e)

### Lecture / Compréhension
| Compétence | Détail |
|-----------|--------|
| Compréhension globale | Identifier le sujet principal d'un texte |
| Compréhension détaillée | Trouver des informations précises |
| Inférence | Déduire ce qui n'est pas dit explicitement |
| Vocabulaire contextuel | Deviner le sens d'un mot grâce au contexte |
| Organisation du texte | Identifier introduction, développement, conclusion |

### Écriture / Rédaction
| Compétence | Détail |
|-----------|--------|
| Structure de phrase | Sujet, verbe, complément |
| Paragraphe cohérent | Idée principale + 2–3 phrases de développement |
| Connecteurs logiques | d'abord, ensuite, finalement, cependant |
| Accord sujet-verbe | Temps principaux : présent, imparfait, passé composé |
| Accord des adjectifs | Genre et nombre |
| Ponctuation | Virgule, point, point d'exclamation, guillemets |

### Grammaire
| Compétence | Détail |
|-----------|--------|
| Classes de mots | Nom, verbe, adjectif, déterminant, pronom |
| Groupes de la phrase | Groupe nominal, groupe verbal |
| Conjugaison | Présent, passé composé, imparfait, futur simple |
| Homophones | a/à, est/et, on/ont, son/sont, ce/se |
| Accord du participe passé | Avec être et avoir (notions de base) |

## Format des Textes de Lecture

Tous les textes de lecture suivent ce format :

```
Titre accrocheur (lié à la mode/beauté)
Illustration / photo d'ambiance
Texte (150–300 mots selon le niveau)
Questions de compréhension (3–5 questions)
Défi vocabulaire (2–3 mots à définir)
```

## Le Journal de Beauté — Ateliers de Rédaction

### Concept
L'enfant tient un **journal de mode/beauté fictif** comme une vraie chroniqueuse.
Chaque entrée de journal est un atelier de rédaction guidé.
Les corrections apparaissent comme des "suggestions de l'éditrice" (jamais en rouge).

## Grammaire Contextualisée

Chaque règle de grammaire est présentée dans le contexte beauté :

### Exemples

**Accord sujet-verbe :**
> "Léa _aime_ le rose. Les filles _adorent_ les paillettes."
> → Pas de règle abstraite. On part de vraies phrases du module.

**Homophones a/à :**
> "Léa **a** un vernis rose. Elle va **à** la boutique."
> → Carte mnémotechnique : "a = avoir (Léa avait), à = direction"

**Temps du passé :**
> "Ce matin, Sophie **a mis** (passé composé) son rouge préféré.
>  Avant, elle **mettait** (imparfait) toujours du rose."
> → Différence expliquée via la routine beauté.

## Système de Correction Douce

| Niveau d'erreur | Affichage | Couleur |
|----------------|-----------|---------|
| Bonne réponse | "Parfait ! Ton éditrice adore !" | Vert menthe |
| Erreur mineure | "Petite retouche : vérifie l'accord de..." | Lavande |
| Erreur majeure | "Ta phrase a du potentiel ! Essaie de..." | Corail doux |
| Hors-sujet | "Belle créativité ! Reviens au sujet : ..." | Pêche |

Jamais de croix rouge. Jamais de "FAUX".
