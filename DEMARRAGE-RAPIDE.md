## Les 3 Innovations Clés

1. **Méthode PALETTE** — 4 étapes colorées pour structurer la résolution de problèmes
2. **Journal de Beauté** — Rédaction via un journal de mode (pas des "devoirs")
3. **Manucure Mentale** — Association couleur-concept pour mémoriser sans effort

## Prochaine action concrète

Commencer par la **Phase 1** :
- Choisir les polices (Nunito + Playfair Display)
- Définir la palette de couleurs (5 couleurs principales)
- Créer la structure de dossiers du projet React
- Écrire les 3 premiers problèmes math scénarisés pour tester le format
